Markdown text editor plugin.

Start

Include markdown.min.js just after the h-t-m-l.min.js script:

<!DOCTYPE html>
<html dir="ltr">
  <head>
    <meta charset="utf-8">
    <title>Test Editor</title>
    <link href="text-editor/ui.i.min.css" rel="stylesheet">
    <link href="text-editor/ui.min.css" rel="stylesheet">
  </head>
  <body>
    <p><textarea id="editor"></textarea></p>
    <script src="text-editor.min.js"></script>
    <script src="text-editor/ui.min.js"></script>
    <script src="text-editor/ui/h-t-m-l.min.js"></script>
    <script src="text-editor/ui/markdown.min.js"></script>
    <script>
    var editor = TE.ui.Markdown(document.querySelector('#editor'));
    </script>
  </body>
</html>

Configure

Instance

TE.ui.Markdown(node, config);
  • node → The text area element.
  • config → Custom configuration data to override the default configuration data.
var editor = TE.ui.Markdown(document.querySelector('#editor'), { … });

Configuration

The available configuration data are:

  • Inherits TE.ui() → read more
  • Inherits TE.ui.HTML() → read more
  • Inherits TE.ui.Markdown()
    • extra → boolean · Enable Markdown Extra features?
    • states → object
      • a → object · List of default links URL and title for referenced–link style.
      • img → object · List of default images URL and title for referenced–image style.
      • abbr → object · List of default abbreviations for abbreviation tool.
    • advance_br → boolean · Press ⇧+↵ to do a hard break?
    • advance_h1,h2 → boolean · Enable SEText header style?
    • advance_pre,code → boolean|string · Replace with true or %1 .foo to enable fenced code block syntax in Markdown Extra.
    • close_h1,h2,h3,h4,h5,h6 → boolean · Replace with true for ### heading 3 ###.
    • close_tr → boolean · Enable closed table pipe?

Keyboard Shortcuts

  • ⌘+b → Bold
  • ⌘+g → Image
  • ⌘+h → Header
  • ⌘+i → Italic
  • ⌘+k → Code
  • ⌘+l → Link
  • ⌘+q → Quote
  • ⌘+r → Horizontal Rule
  • ⌘+t → Table
  • ⌘++ → List
  • ⌘+- → Bullet
  • ⌘+✘ → Strike
  • ⌘+↓ → Footnote
  • ⌘+↑ → Footnote
  • ⌘+↵ → Paragraph
  • ⌘+⇧+? → Abbreviation
  • ⇧+↵ → Hard Break
  • ⇥ → Indent & Depends
  • ⇧+⇥ → Outdent
  • ↵ → Depends
  • ⌫ → Depends
